Attendance Approval and Reject Workflow in Odoo

Approve and Reject Attendance in Odoo

This app allows your manager to approve and reject workflow on Attendance form view.


Main Features:

  • Allow the manager to approve attendance as shown.
  • Allow the manager to reject attendance with reason as shown.
  • Allow them to approve and reject attendance from a list view for multiple attendance records together.
  • Allow them to approve and reject multiple attendances from the mass approve attendance and mass reject attendance button.
  • For more details please check below screenshots and watch the video.
