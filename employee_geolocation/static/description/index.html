The User Location module allows the admin to track the time and location when an employee checks in and out.

For this module the admin can make the following options:

  • The employee can see basic data of an attendance.
  • Admin can get the access to view the login details of employee.
  • The login details will consist- location (Name of the city, State and Country), Date and time.
  • You can see the history of an employee according to data from an attendance.
  • Identify and group the device and browser types.
  • Identify and group the morning and afternoon sessions.
  • See on Google map the locations registered with an assistance.

This module has a dependency of: Geospatial support for Odoo
